# Curso de IA

Repositorio de materiales para un curso introductorio orientado a construir productos con inteligencia artificial.

## Estructura

- `sesiones/`: materiales teoricos por sesion.
- `programa/`: programa general, version alternativa 2026 y silabus.
- `assets/`: portadas e imagenes usadas por los documentos Markdown.
- `mcp-openai-images/`: herramienta tecnica para generar portadas desde los archivos del curso.

## Orden recomendado de lectura

1. `programa/programa-curso-ia.md`
2. `programa/silabus-curso-ia-primera-version.md`
3. `sesiones/material-sesion-01-fundamentos-ia-llms.md`
4. `sesiones/material-sesion-02-redes-neuronales-transformers-entrenamiento.md`
5. `sesiones/material-sesion-03-prompting-chat-asistentes.md`
6. `programa/programa-curso-ia-2026-alternativo.md`

## Nota sobre la carpeta tecnica

La carpeta `mcp-openai-images/` no forma parte del contenido de clase. Es un proyecto auxiliar para generar portadas e imagenes del curso a partir de los documentos Markdown.
